# Floatwatch V2.3

V2.3 轻量悬浮设置版。

更新重点：
- 新抽象 LOGO：弧形时间刻度 + 指针 + 单圆点，减少元素。
- 首页底部保留「设置」「开启悬浮时钟」。
- 设置页只保留：自动刷新（1s/3s/5s）、悬浮样式（精简模式/小号完整模式）、深色/浅色、透明度。
- 悬浮时钟设置页参考紧凑样式：
  - 时钟模式：时间偏移 + 刷新延迟
  - 倒计时模式：结束时间、预设时间（30s/1min/3min/5min）、时间偏移 + 刷新延迟
- 倒计时在未选择结束时间和预设时不可开启。
- 延迟刷新改为小图标/单次刷新，取消“精准测试”入口。
- 延迟测试内部改为静默三次取优，页面只显示结果。
